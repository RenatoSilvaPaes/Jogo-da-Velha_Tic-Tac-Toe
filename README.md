# Jogo da Velha/Tic Tac Toe
Português: Jogo da Velha em Python. Este script possue 3 modos de jogo: usuário contra usuário; usuário contra computador; e computador contra computador. Caso o usuário escolha entre uma das opções que tenha um jogador máquina, ele terá que escolher a dificuldade dela. As dificuldades são: FÁCIL, a qual o computador só escolhe uma casa aleatória que esteja disponível no tabuleiro; MÉDIO, a qual o computador tenta fazer a combinação de casas para poder vencer o jogo; e DIFÍCIL, a qual o computador verifica se o seu adversário (usuário ou máquina) precisa de pelo menos UMA casa para vencer. Se for o caso, ele impede sua vitória.

English: Tic Tac Toe in Python. This script has 3 game modes: user versus user; user versus computer; computer versus computers. If the user chooses between ona the options that has a machine player, he needs to choose It's difficulty. They are: EASY, which the computer only chooses available houses on the board randomly; MEDIUM, which the computer tries to make the sequel of 3 houses to win; and HARD, which the computer verifies if It's adversary (user or machine) needs at least ONE house to win. If that's tha case, It will prevent It's adversary's victory.
